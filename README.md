# TrendFit

## Description
    TrendFit is a smart fashion recommendation system that helps ,<BR>
    users choose clothes based on their skin tone, body measurements, and personal preferences.<BR>
    It uses artificial intelligence to provide personalized outfit suggestions, making fashion selection easier and more accurate.<BR>
    The system also includes features like size prediction and outfit combination analysis to improve the overall shopping experience.

## Features 

    - Personalized outfit recommendations
    - Skin tone detection using K-Means clustering
    - Outfit combination using CNN
    - AI-based body measurement for size prediction
    - User-friendly web and mobile interface
    - Add to cart and secure payment system
    - Upload custom dress designs
    - Reviews and feedback system
    - Complaint management
    - Admin dashboard for managing products and orders
    - Secure login and authentication
    - Responsive design
  

## Technologies 

    - Frontend: HTML, CSS, JavaScript, Dart 
    - Backend: Python Django framework
    - Database: Mysql
    - AI/ML: K-Means Clustering, CNN, OpenPose,Face detect, allotskintone
